About the theme

在 Github 上逛的时候发现了一个非常简洁的主题 leedom 自己在他的基础上修改了一下，把 ejs 重构到 pug（原jade） 因为比较方便

  • 增加了 giscus 评论
  • 重构代码，把 posts, tags, archive 页的代码解耦，提取公共 layout.pug
  • 增加了 category 分类页面
  • 兼容 hexo-generator-category-enhance, 无需手动创建 category 页面

修改后放在 Github, 取名叫 leek 😄
